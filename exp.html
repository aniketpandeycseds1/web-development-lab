Course Curriculum

Objectives

1. To design a simple course curriculum webpage using HTML.
2. To practice the use of basic text formatting tags like bold, italic, and underline.
3. To learn how to make ordered, unordered, description, nested, and mixed lists.
4. To understand webpage content structuring using headings and paragraphs.

Tools Required

  • Computer or Laptop
  • Text Editor (VS Code, Notepad++)
  • Web Browser (Chrome, Edge, Firefox)

Theory

1. Headings

Used to display titles and subheadings. Example: <h1> to <h6>.

2. Paragraphs

Used to define blocks of text. Example: <p>Paragraph Text</p>

3. Formatting Tags

<b> for bold text
<i> for italic text
<u> for underlined text
Special characters are shown using HTML entities like ©, <, >, &.
